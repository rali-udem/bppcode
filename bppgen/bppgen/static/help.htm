This application allows head-hunters to generate recruitment letters destined to be sent to potential candidates. It uses the information contained in a job offer and a candidate profile to create a letter tailored to the candidate.

  1. Click Start
  2. Enter the job offer. To do this, you have two options.
    1. You click the pencil icon and fill out the fields, or
    2. You enter an offer id (retrievable in the backing database, e.g. 55157fc5c51c771c4bc36ffe ) or a keyword to be found in an offer, then click the button Load offer. [See Note below]
  3. Click the button Next
  4. Select a candidate by clicking the button New candidate. Like for the offers, you have two options to fill out the information related to a candidate.
    1. You fill out the fields manually.
    2. You enter a candidate id (retrievable in the backing database, e.g. 52b31a880b045119318b45a0 ) or a keyword to be found in a profile, then click the button Load profile. [See Note below]
  5. Click the button Add to list
  6. Click the button Generate. This generates the letter after a few seconds.

Note: Retrieving an offer or a candidate from the underlying databases is only possible if the bridge between the application and these databases is correctly set up. See the instructions provided with this package.
